# Boss 求职助手

AI 甄别 Boss 直聘消息，自动回复 + 发简历的 Chrome 扩展。

纯浏览器运行，无需本地服务。

## 功能

- **多 LLM 支持** — 智谱 GLM / OpenAI / Claude / Gemini / MiniMax，配置面板一键切换
- **AI 岗位评分** — 薪资/城市/行业/公司/方向多维度加权，0-200+ 分，≥70 自动投递
- **自动投递** — 达标岗位自动发送招呼语 + 在线简历，失败自动重试
- **面试检测** — 关键词 + LLM 语义双层识别，面试邀请跳过自动回复
- **婉拒识别** — 28 个婉拒关键词预筛 + LLM 判别，节省 API 额度
- **二次回复** — 已投递会话的新消息自动礼貌回复（每天最多一次）
- **飞书通知** — 通过飞书应用 API 推送投递/面试/告警通知（可选）
- **实时进度** — 扫描进度条、轮次统计、等待倒计时
- **调试模式** — 模拟扫描不实际发送，不污染记录
- **缓存管理** — 一键清除处理记录，重新扫描

## 安装

1. 下载或 clone 本仓库
2. Chrome 地址栏输入 `chrome://extensions`
3. 开启右上角**开发者模式**
4. 点击**加载已解压的扩展程序** → 选择本目录
5. 点击扩展图标 → **配置** → 填写 API Key 和个人信息 → **保存配置**

## 使用

1. 打开 [Boss直聘消息页](https://www.zhipin.com/web/geek/chat)
2. 点击扩展图标 → **开始扫描**
3. 扩展每 30 秒自动扫描一轮

### 按钮说明

| 按钮 | 功能 |
|------|------|
| 开始扫描 / 暂停扫描 | 启停自动扫描循环 |
| 立即扫描一轮 | 手动触发单次扫描 |
| 调试模式 | 模拟运行，不实际发送消息 |
| 查看统计 | 显示投递/面试/婉拒/跳过统计和最近记录 |
| 清除缓存 | 清空所有处理记录，可重新扫描 |
| 配置 | 打开配置面板 |

## 配置项

### 身份信息
- 姓名、现任岗位、现任公司、经验年限、技术栈

### 筛选条件
- **薪资基准线**（年薪 K）— 等于基准 = 50 分，高于加分低于扣分
- **自动投递阈值** — 评分达到此值自动投递（默认 70）
- **目标城市** — 逗号分隔
- **黑名单行业** — 逗号分隔
- **每轮最大处理数** — 控制单轮扫描量

### AI 模型
- 提供商选择（GLM/OpenAI/Claude/Gemini/MiniMax）
- API URL（自动填充，可自定义）
- API Key
- 模型选择（支持自定义模型名）

### 飞书通知（可选）
- 启用 / 关闭
- App ID / App Secret（飞书自建应用凭证）
- 接收者 Open ID
- 通知场景：自动投递 / 面试邀请 / 轮次摘要

### 连接测试
- **测试 AI 模型连接** — 使用当前配置实际调用 LLM，验证 API Key 和网络连通性
- **测试飞书通知** — 发送一条真实测试消息到飞书，验证应用凭证和接收者配置

## 评分规则

以下为**默认值**，所有分值均可在扩展配置面板「评分调节」中自定义修改。

| 维度 | 默认分值 | 说明 |
|------|----------|------|
| 薪资（核心） | 基准线 = 50 分 | 高于 +1/5K，低于 -1/10K，基准线可配 |
| 非目标城市 | -20 | 在目标城市列表中不扣分 |
| AI 方向匹配 | +10 | 岗位涉及 AI/Agent/LLM 等 |
| 非 AI 方向 | -5 | 与 AI 无关的岗位 |
| 知名公司 | +10 | BAT、字节、华为等 |
| 上市公司 | +5 | A 股/港股/纳斯达克 |
| 猎头/HR 联系 | +5 | 主动联系加分 |
| 996/大小周 | -20 | — |
| 外包/外派 | -100 | — |
| 信息不足 | -50 | 无薪资无岗位信息 |
| 垃圾/诈骗 | -1000 | 副业、刷单、传销等 |
| 黑名单行业 | -1000 | 博彩、P2P 等 |

最终 score = max(0, 各项之和)

## 隐私说明

- 所有数据存储在浏览器本地（`chrome.storage.local`），不上传任何服务器
- LLM API 调用仅发送岗位摘要信息（公司名/岗位/薪资/消息摘要），不发送个人简历内容
- API Key 仅存储在浏览器本地存储中
